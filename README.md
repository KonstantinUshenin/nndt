## Install
 
```
pip install git+https://github.com/KonstantinUshenin/nndt.git
```

## Test

How to run the tests?
 - Unpack `./test/acdc_for_test.7z`.
 - Configure your IDE to run test from `test` folder and assume `nndt/nndt` as a source root.
 - Run test;)
